# **App Name**: CodeGenius

## Core Features:

- Prompt-Based Website Generation: Generate a structured website layout with CSS, sections, and placeholder images based on user prompts. Includes navigation and footer generation.
- Real-Time Customization Panel: Allow users to customize color palettes (preset + custom HEX), font families (Google Fonts), and website types (portfolio, business, blog, landing page, ecommerce mockup, agency/SaaS marketing page) with live previews.
- Template Gallery: Provide a gallery of pre-built templates (Modern Minimal, Vibrant Colorful, Dark Elegant, Corporate, Tech/SaaS, Artistic Portfolio, Ecommerce Landing) with a one-click "Load Template" option.
- Real-Time Preview: Display a real-time website preview that automatically updates as users modify the prompt, colors, fonts, and template.
- Export Options: Enable users to export the full project as a ZIP file, individual HTML/CSS/JS files, a JSON config, or the template only.
- AI-Powered Enhancements: Include AI-driven features such as auto-generating responsive design and royalty-free placeholder images. The feature also supports an option to generate images and website text using AI, auto-generating SEO meta tags.
- AI "Fix Design" Tool: AI-powered tool to automatically adjust colors and spacing based on design principles. Additionally provide drag-and-drop reordering sections.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to convey calmness and innovation.
- Background color: Very light gray (#F5F5F5) to maintain a clean, modern backdrop.
- Accent color: Muted violet (#8A2BE2) to highlight key interactive elements.
- Headline font: 'Space Grotesk' sans-serif for headlines and short amounts of body text; longer text should be 'Inter'.
- Use simple, modern icons to represent website types and functionalities.
- Implement a clean, intuitive layout with a clear step-by-step flow, inspired by Framer and Webflow.
- Incorporate smooth, subtle animations to enhance user experience and provide feedback.
